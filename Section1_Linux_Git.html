Q2: Why is SSH key-based authentication more secure than password authentication?

Answer (English):

SSH key-based authentication uses cryptographic key pairs (public/private), making it nearly impossible to brute-force. Keys are long, randomly generated strings that cannot be guessed. Passwords can be weak, reused, or cracked through brute-force attacks.

Q3: What is the difference between the root user and a sudo user?

Answer (English):

The root user has unlimited system privileges and can do anything without restrictions. A sudo user is a regular user who can temporarily gain root privileges using the `sudo` command. This provides better security through the principle of least privilege and audit trails.

Q4: Why is a firewall required on a server?

Answer (English):

A firewall acts as a security barrier, controlling incoming and outgoing network traffic. It blocks unauthorized access attempts, prevents attacks, and follows the defense-in-depth security principle. Without a firewall, all ports are open, making the server vulnerable to attacks.
